# 라디언스 캐시 바운스-필 (실내 간접광 품질) — 계획

## 동기 / 현재 상태
실내 스카이라이트 차폐(`gi-radiance-cache.md` v3)는 **차폐된 IBL diffuse를 평면 중립 틴트**
(`P_SKYVIS_TINT`, OcclusionTint형)로 채운다. 이건 표준 레퍼런스 메커니즘(틴트 = 바닥값, 간접
바운스가 나머지를 채움)이지만, 우리의 채움은 **평면(무채색·무방향)**이라 차폐 영역에 **색 있는
바운스광**(예: 빨강 커튼이 인접 벽/바닥에 번지는 빛)이 빠진다. 목표 = 차폐분을 **라디언스
캐시(GI 볼륨)의 실제 바운스 E**로 채워, 차폐 영역이 평면 회색이 아니라 **국소적으로 채색된 간접광**을
받게 한다(완전 레퍼런스형 실내 GI).

## 막힌 지점 (측정으로 확정, 2026-07-01 — S1 정밀 진단 완료)
바운스-필을 그대로 시도하면 **닫힌 중앙 바닥이 검정 [0,0,0]**. 정밀 진단으로 **두 개의 구조적 원인** 확정:

**원인 A — AABB가 바닥 아래로 3m 확장 → 지하 프로브 오염.** `sponza_intel` scene_aabb =
`y∈[-3.12, +20.75]`(높이 23.87m). 32 프로브 → **y-간격 0.746m**. 바닥(y=0)은 프로브 4(y=-0.13,
**지하**)와 프로브 5(y=+0.61, 실내) 사이, **uvw.y=0.131 → 프로브 4.18**(지하 프로브에 82% 가중).
지하 프로브는 "지오메트리 내부"라 모든 레이가 즉시 hit/degenerate → **radiance E≈0 AND sky-vis V≈0**.
**★ 역설**: 바닥이 V≈0(지하 프로브)을 읽어 **강하게 차폐 = v2의 디블루가 잘 되는 이유**. DDGI 노멀
오프셋으로 "고치면" 실내 프로브를 읽어 **V 상승 → 차폐 약화 → 디블루 사라짐**(v3b에서 rb −36→−50로
**더 파래진** 게 이것). 즉 표준 S1 픽스가 v2의 좋은 결과와 **충돌**한다.

**원인 B — 열린 중앙은 진짜로 하늘을 봄(바운스가 물리적으로 ~0).** `sky_fill`을 0.4→8.0(20×)로 올려도
중앙 E 변화 없음 + rb E도 불변([33.7]; rb는 **태양-바운스 지배**라 sky_fill 무관). 열린 홀 중앙 프로브는
레이가 대부분 하늘로 escape(miss=0) → 채색 바운스가 애초에 없음. → 중앙 바닥의 중립 틴트는 **물리적으로 옳음**.

**결론: v2(틴트 + 전역 가중 바운스 E)가 이미 바운스-필의 실효 = 거친 32³의 실용 최적.** E는 이미 어디서나
더해지고(rb 등 커튼 인접면에서 채색 기여), 중앙은 진짜 바운스가 없어 중립이 옳다. S1의 표준 픽스(오프셋)는
디블루를 깬다. → **실질 개선은 S4(촘촘한 카메라-추종 프로브 클립맵)뿐**이며, 그것이 이 트랙의 핵심 선결.
S2/S3는 S4(유효 프로브 밀도) 없이는 무의미/역효과.

## 작업 스테이지 (S1 진단 후 재정렬)

> **진단 결과 S0가 선결로 승격.** 위 원인 A/B가 보여주듯, 거친 32³ 단일 그리드로는 (지하 프로브 오염 +
> 열린 중앙) 의미 있는 바운스-필이 불가하고, 표준 오프셋 픽스는 v2 디블루를 깬다. **유효 프로브 밀도/배치**
> (S0)가 없으면 S1~S3는 무의미하거나 역효과. v2는 그대로 출하 상태로 두고, 이 트랙은 S0부터 새로 쌓는다.

## 레퍼런스 정렬 (상용 엔진 월드-프로브 irradiance-field 소스 대조, 2026-07-01)
상용 엔진의 월드-공간 irradiance 프로브 필드(우리 GI 볼륨의 직접 대응물)를 소스 대조해 **기본값·동작방식**을
추출했다(상표명은 규칙대로 표기 안 함). 핵심은 **정규 프로브 그리드를 실내에서 쓸 수 있게 만드는 4가지 기법**:

1. **카메라-추종 클립맵** — N레벨(기본 **4**), 레벨마다 worldExtent ×base(기본 **2.0**) → 근거리 고밀도·원거리
   광역. 레벨당 그리드 해상도 기본 **64³**, 최내곽 extent 기본 **50 m** → 최내곽 프로브 간격 ≈ **0.78 m**.
   프로브당 방사 octahedral **16²**, irradiance octahedral **6²**(우리 SH-L1과 동급의 저주파).
2. **★ 프로브당 깊이(occlusion) + 체비셰프 가시성 보간** — 프로브당 거리 통계(평균·평균²)를 octahedral **16²**로
   저장. 리시버가 주변 8프로브를 **하드웨어 trilinear 대신 가시성 가중**으로 보간: 리시버→프로브 거리를 프로브가
   그 방향으로 저장한 깊이와 비교(체비셰프) → **표면 뒤(=우리의 지하 프로브)·벽 너머 프로브는 가중 ~0**.
   **이게 우리의 지하-프로브 오염과 누설을 동시에 푸는 핵심.** occlusion bias 기본 **0.8**.
3. **프로브 재배치(relocation/offset)** — 프로브를 솔리드 밖으로 약간 이동(probe world offset). 기본 **off**
   (체비셰프만으로 충분); opt-in.
4. **프로브 마킹 + 아틀라스 저장** — 화면이 필요로 하는 프로브만 트레이스(downsample 기본 **2**), 프로브를 2D
   아틀라스(기본 **128² 프로브**)에 두고 grid→atlas 인디렉션 텍스처로 매핑(클립맵 전체를 메모리 바운드로).

**우리 제약**: R32F 볼륨·64 볼륨 슬롯·SH-L1·GDF SW-RT 트레이스. 아틀라스+인디렉션 전면 도입은 무겁다 → 아래처럼
**체비셰프 가시성(2번)을 우선** 도입해 기존 32³ 그리드를 실내-유효화하고, 클립맵/아틀라스(1·4번)는 후순위.

> **★ S0a 구현·측정 완료 (2026-07-01) → 이 씬에선 INERT, 되돌림.** 거리 모멘트 생산(`gi_volume.slang`,
> +4볼륨) + 수동 8-프로브 above-ground/체비셰프 가중 소비(`gdf_gi.slang`) + `P_GI_VOL_CHEBYSHEV`(기본 off)
> 전부 구현. **default off = v2 바이트 동일**(무회귀). **그러나 ON ≈ v2**(바닥 박스 ±0.2/ch). 측정으로
> 드러난 정정: 원인 A(지하 프로브)는 **오진**이었다 — 바닥 아래 프로브(y=-0.13)는 degenerate가 아니라
> **상반구로 방을 정상 트레이스**한다(하반구만 즉시 ground hit). 진짜 원인은 **B(열린 중앙이 하늘을 봄 →
> E≈0)**뿐이고, 그건 프로브 *선택*으로 못 고친다(읽을 바운스가 없음). 누설도 이 씬엔 없어 체비셰프도 inert.
> → S0a는 **inert + 256B 푸시(Vulkan 한계, DX≡VK 검증 불가)** 라 규칙대로 되돌림. **결론 재확인: v2 최적,
> bounce-fill 트랙은 원인 B(아키텍처/물리)로 막힘.** 더 가려면 화면-공간 프로브(대공사, 실내 바닥엔 한계효용).

### S0a (핵심·선결) — 프로브당 깊이 + 체비셰프 가시성 보간 (지하-오염·누설 동시 해결)
기존 32³ 그리드 유지. 프로브당 **거리 모멘트**(평균 dist, 평균 dist²)를 추가 저장하고, 리시버 소비를 하드웨어
trilinear에서 **수동 8-프로브 체비셰프 가중 보간**으로 교체.
- 생산(`gi_volume.slang`): 각 프로브가 레이 hit까지의 거리로 mean/mean²를 누적(방향성은 우선 스칼라/SH-L1로
  근사) → 스칼라 R32F **2볼륨/슬롯**(×2 ping-pong = 4볼륨; 총 GI 32+4=36 ≤ 64). EMA.
- 소비(`gdf_gi.slang`): vol 분기에서 `world_pos`를 감싸는 8프로브를 명시 순회, 각 프로브 방향 깊이와 리시버
  거리로 체비셰프 가중 `w = (d≤mean) ? 1 : variance/(variance+(d−mean)²)` × trilinear → 가중합으로 E·V 재구성.
- **이게 통과하면 v2의 "지하 프로브 V≈0 덕에 디블루" 우연을 정식 가시성으로 대체** → 바닥이 유효 실내 프로브를
  읽되 가시성으로 올바르게 차폐(디블루 유지가 **원리적**으로). 표준 노멀-오프셋(작은 바이어스)도 이 위에서만 적용.
- 게이트: 바닥이 지하 프로브 0% 가중 + **인접 다른-색 방 색 누설 0** + 갤러리 바이트 동일 + 결정론적.

### S0b (선택) — 프로브 재배치 (솔리드 밖으로 offset)
S0a로 부족하면 프로브를 가장 가까운 빈 공간으로 약간 이동(per-probe world offset 볼륨 3채널). 기본 off.

### S0c (후순위, 큰 작업) — 카메라-추종 클립맵 + 아틀라스 저장
밀도가 더 필요하면(작은 디테일) 2~4레벨 클립맵 + 프로브 아틀라스/인디렉션으로 저장 재설계. 메모리·인덱싱 대공사 →
S0a/b가 불충분함을 **측정으로 입증한 뒤에만**.

### S1 — 볼륨이 실제 스카이라이트 바운스를 운반 (시딩/전파)
차폐 영역을 채우려면 볼륨 E가 **바운스된 하늘광**을 닫힌 중앙까지 운반해야 한다.
- **S1a 하늘 시딩**: 프로브 hit 셰이딩의 상수 `sky_fill`을 **실제 diffuse 하늘 irradiance**로 교체
  (IBL irradiance 큐브를 compute에서 샘플, 또는 sky.slang 기반 SH를 CPU 업로드해 큐브-독립적으로).
  ⚠ compute에서 큐브 샘플이 0이 아닌지 먼저 **격리 검증**(작은 디버그 패스로 한 voxel 출력) — residency는
  영속이라 OK일 가능성이 높으나 ping-pong read/write 인덱스 정합을 확인.
- **S1b 전파 수렴**: 32³ 멀티바운스가 닫힌 중앙까지 도달하도록 — EMA α/워밍업 프레임 상향, 또는 프로브당
  멀티바운스 샘플 수 증가. 측정: DEBUG_VIEW=10에서 **중앙 바닥 E > 0** 달성이 S1 통과 기준.
- 게이트: 갤러리 바이트 동일(콘텐츠 전용), 결정론적.

### S2 — (S0a로 흡수됨) 프로브 가시성/누설 차단
체비셰프 가시성은 위 **S0a**에 통합. 별도 스테이지 불필요.

### S3 — 차폐분을 바운스로 필 (틴트 → E)
S1·S2 통과 후, `pbr.slang`에서 차폐분 채움을 평면 틴트에서 **E(라디언스 캐시 바운스)**로 교체:
`diffuse = albedo·(irradiance·V + E_bounce·(1−V))` (이중계상 없음: V는 하늘 가시분, E는 바운스분).
⚠ 기존 `gi` 항(E를 전역 가중 1로 더함)을 (1−V) 가중으로 옮기되 **갤러리(가시성 미바인드, V=1)에서
(1−V)=0이 GI를 죽이지 않도록** 분기/센티넬 처리(갤러리는 현 레이마치 경로 유지). `P_SKYVIS_TINT`는
**안전 바닥**으로 남김(작은 기본값).
- 측정: 닫힌 바닥이 검정 아님 + **커튼 인접 바닥/벽에 색 바운스 가시**(R−B가 평면-틴트보다 채색적으로 상승).
- 게이트: 갤러리 바이트 동일, 결정론적, off-스위치 유지, 패스트레이서 잔차 비교(`P8_PATHTRACE`)로 과/소노출 확인.

### S4 — (S0c로 이동됨) 프로브 밀도 / 카메라 클립맵
클립맵/아틀라스는 위 **S0c**로. 측정이 요구할 때만.

## 비용 / 리스크
- 소비 비용 증가: S0a는 하드웨어 trilinear → **수동 8-프로브 체비셰프 보간**(프로브당 추가 거리-모멘트 read).
  half-res GI에선 픽셀당 적음. `PROFILE_GPU` 측정. S0c(클립맵/아틀라스)가 진짜 비용·복잡도.
- 주 리스크: 체비셰프 바이어스 튜닝(과하면 과차폐, 약하면 누설 잔존), S3 과/소노출 → 패스트레이서 잔차로 정직 보고.
- 전부 콘텐츠 전용(`gi_volume` ON일 때만) → 갤러리 앵커 보존. DX≡VK는 Windows 게이트(현 동결 시 Metal 구현+보류 명시).

## 진행 순서 (권고)
**S0a(체비셰프 가시성) → S3(틴트→바운스 필) → 측정**. 여기까지가 트랙의 핵심 가치(원리적 디블루 + 색 바운스 필).
S0b(재배치)·S0c(클립맵)·S1(하늘 시딩)은 측정이 부족함을 입증할 때만. 각 단계 갤러리 바이트 동일·결정론적·
off-스위치 유지가 게이트.

## 하지 말 것
- 갤러리 앵커 깨기. 평면 틴트를 성급히 제거(S0a 가시성 전엔 검정/지하-오염 회귀). 측정(지하 0% 가중, 누설 0) 없이 S3 진입.
- 거친 32³를 그대로 두고 디테일을 기대(S0c 없이는 저주파). 이중계상(V·irradiance + E를 둘 다 전역 가중).
- 하드웨어 trilinear로 프로브 보간(가시성 무시 → 지하/누설). S0a는 반드시 수동 8-프로브.
