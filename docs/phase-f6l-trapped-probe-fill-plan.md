# F6L 계획서 — 히어로 블랙-박스: 갇힌-프로브 충전 + bent-normal 노이즈 게이트

> 상태: 구현 완료·검증 중 (2026-07-20). F6I §0 재개 조건 ②(볼륨-공간 평활 — **증상
> 완화**)로 승인받아 열었으나, 조사 결과 **박스의 진범은 별개 버그(bent-normal 노이즈
> 정규화, §3)** 로 판명 — 충전(§1)은 E-측 실결함 2건의 수리로 함께 랜딩. §0의
> 재시도-금지 원장(이진-부호 전 계열·해상도·τ-radiance 등)은 그대로 유효 — 본 페이즈는
> 그 어떤 항목도 재시도하지 않는다.

## 0. 동기 (실측)

`sponza_hero` 히어로 재캡처(인게임 기본 스택)에서 **사이프러스 캐노피 뒤 커튼/잎에
~2–3 m 검은 박스**가 표면화. 격리 실측(전부 단일-차이 A/B, EV11 고정노출):

- 알베도/직접광 뷰 정상, `P_GI_VOLUME=0`에서만 소멸 → **볼륨 GI 필드 자체의 결함**.
- `P_GI_VOL_CLIP=0`(fine off) 불변 → **코스 레벨 셀**. `P_GI_VOL_OCC=0`에서 형상 변형
  (점유-게이팅이 해당 셀과 상호작용) — 소비측이 아니라 **필드 값이 죽어 있음**.
- 기전: 밀집 캐노피(4.9 M-tri ivy/사이프러스류) **내부**의 코스 프로브는 전 방향이
  근거리 히트 → V≈0, sky_fill×V(히트)≈0, 멀티바운스 피드백도 자기-강화 0 →
  **E≈0/V≈0으로 결정론적 수렴**(F6I §0 판독 규칙의 "프로브-갇힘 지문" 그대로).
  트라이리니어 풋프린트(±1 복셀 ≈ 코스 1 m급)가 인접 표면에 박스로 투영된다.
- 7/1 히어로에 없던 이유: 당시 볼륨 필드는 Metal 디스패치 버그로 3/4 죽어 있었음
  (gi-volume-leak 페이즈) — 필드가 살아나면서 이 클래스가 노출된 것.

## 1. 설계 — §2m "3³ 블러"의 안전한 인스턴스화

사전 등록된 형태(새 컴퓨트 패스 + 전면 3³ 커널)를 그대로 두지 않고 **갇힌-셀 한정
충전**으로 좁힌다. 근거:

1. **bindless 천장**: 표시용 볼륨 세트(12+4장) 추가는 volumes[64] ~49슬롯 사용
   상태에서 불가. 별도 패스는 표시용 세트 없이는 핑퐁 피드백에 블러가 유입된다.
2. **확산 위험**: EMA 피드백 루프 안의 전면 블러는 매 프레임 필드를 번지게 하는
   확산 PDE — 건강한 영역의 콘트라스트(커튼 색-번짐의 방향성)까지 잠식.
3. **게이트 안전**: 지문(V≈0 ∧ E≈0-상대)에 걸린 죽은 셀만 변경 — 건강한 셀은
   추정기·EMA·저장까지 비트-동일. `P_GI_TRAPPED_FILL=0`은 전 경로 레거시 복원.

### 기전 (gi_volume.slang, 추정기 직후·EMA 직전)

- **지문 판정**(프레임 자체 추정치로): 구면 평균 가시도 `V̄ = sv0·kw·SH_C0 < 0.02`
  이면서 자기 E 휘도가 이웃 평균의 5% 미만.
- **이웃**: 자기 레벨의 6-면 이웃 텍셀(레벨 경계 클램프), **이전 프레임 슬롯**에서
  포인트 리드(핑퐁 read 슬롯 = 완성된 필드 — 동기화 불필요, 결정론 유지).
- **벽-투과 차단**: 프로브↔이웃 **중점의 `scene_occ` < 0(지오메트리 내부)이면 그
  이웃 기각**. 두꺼운 벽 안 셀은 이웃이 전부 기각돼 충전 자체가 일어나지 않고(현행
  유지), 다공성 캐노피만 중점이 열려 있어 충전된다 — 원하는 판별이 공짜로 나온다.
- **충전**: 채택 이웃들의 radiance SH 4계수 평균으로 이 프레임 추정치를 대체.
  **sky-vis(V)는 불변** — 캐노피 내부의 V≈0은 물리적으로 참이며, 결함의 주범은
  소비측 앰비언트의 E 항. 대체 후 EMA는 정상 진행(자기 이력과 블렌드 — 다중
  프레임에 걸쳐 이웃 평균으로 수렴, 슬랩 주기당 한 셸씩 안쪽으로 전파).
- **플래그**: `GiVolumePush.flags` bit3(`P_GI_TRAPPED_FILL`, 기본 ON), push 크기 불변.
  fine 레벨에도 동일 적용 — §2m의 "코스 한정"은 **블러의** 콘트라스트 잠식을 막기
  위한 한정이었고, 충전은 죽은 셀만 만지므로 그 우려가 구조적으로 없다(카메라가
  캐노피에 접근해 fine 박스가 나무를 덮으면 동일 결함이 fine에서 재발하므로 포함).

## 2. 격리 원장 (실측 — 전부 단일-차이 A/B, EV11 고정노출·WARMUP 64)

**박스 불변(기각)**: `P_GI_VOL_CLIP=0`(fine) · `P_CACHE_LIT_CALIB=0` ·
`P_GI_READ_OFFSET=0` · `P_REFLECT_PROBE_VALID=0` · `P_CACHE_SKY_OCCLUDE=0` ·
`P11_GDF_REFLECT=0`(SW 반사 전체 off!) — 반사 트레이스 경로 전체 무죄.
`P11_CACHE_STREAM`/`F5_PERMESH_ALBEDO`는 기본 off라 A/B 무효(교훈: env_bool 기본값
확인 후 A/B). `P_GI_VOL_OCC=0`은 **형상만 변형**(점유-게이팅이 셀 경계를 날카롭게 함).

**박스 소멸**: `P_GI_VOLUME=0`(볼륨 경로 전환 — 원인 아닌 착시) ·
**`P_BENT_NORMAL=0`(진범 지문 — §3)**.

**디버그-뷰 판정**: 알베도(1)·direct(7)·AO(6/9) 무죄 / **raw E(뷰15) 매끈 — 박스
없음** / 뷰8(앰비언트)에 박스 선명 → 디퓨즈 스카이라이트 항으로 수렴.

## 3. 진범 — bent-normal 정규화의 신뢰도 파괴 → 유효도 계약 (랜딩)

sky-vis 밴드-1 벡터(공짜 bent normal)는 **정규화가 크기(=신뢰도)를 파괴**한다: 평균
V≈0 셀의 밴드-1은 추정 노이즈인데 구 1e-5 길이 엡실론이 이를 풀-강도 단위 벡터로
정규화했고, 소비자(`occlude_sky_diffuse_bent`)는 **V가 낮을수록 그 방향을 강하게
신뢰**(`w=V`)한다 — 셀-양자화된 dot_factor가 복셀-샤프 암흑 박스로 표면화(사이프러스
캐노피 V≈0 셀군 → 뒤 커튼).

**수리 반복 원장(전부 실측, 단일-차이)**:
| 변형 | sunlit(예산 19.2) | 박스 |
|---|---|---|
| 봉인 기준선(버그 재현) | 19.101 | **선명(크리스프 직각)** |
| 프로듀서 제로잉(V<1%→bent=0→스칼라 밝음) | 19.882 ✗ | 소멸 |
| 소비측 기대값 1/4 균일 치환 | 19.647 ✗ | 소멸 |
| dot→0 암전 치환 | 19.651 ✗ | 소멸 |
| leak lum 등방-평균(대척점 2-탭) 추가 | 19.639 ✗ | 소멸 |
| V-플로어 1e-3 축소 | 19.336 ✗ | 부분 재발 |
| **유효도 계약(랜딩)** | **19.159 ✓** | 크리스프 모서리 제거·연화 |

핵심 발견 2건: ① 상쇄-착지 — 노이즈 dot_factor가 V≈0 픽셀의 스카이라이트 과주입을
통계적으로 억누르고 있었고 봉인 캘리브가 거기 적합돼 있다(§0 판독 규칙 그대로).
② **젠센 위반 = 신호 검출** — 평균/암전 치환이 전부 게이트를 악화시키는 것은 낮은-V
밴드-1이 순수 노이즈가 아니라 **실제 개구 방향 신호**를 담고 있다는 증명.

**랜딩된 계약**: 프로듀서(gdf_gi bent 4사이트)가 **방향 × 유효도**를 기록 — 유효도 =
`sat(√3·c0/|b|)` (**SH 비음성 경계**: 실제 V(ω)는 |밴드-1| ≤ √3·밴드-0, 점-개구에서
등호 — 물리적으로 가능한 방향은 전부 1 = 레거시 비트-근사-불변, 위반=노이즈만 0으로).
스케일-불변(재정규화 인자 상쇄)·무튜닝. fine/coarse lerp 사이트는 재정규화 제거(유효도
선형 블렌드가 올바른 의미). 소비자 2곳은 |bent|를 신뢰도로 읽어 bent-프리 계약으로
페이드(`occlude_sky_diffuse_bent` — per-pixel 스칼라 경로 / `spec_occlusion_bent` —
no-op). 갤러리 bent==0 계약 불변.

## 4. 게이트·정책 (최종)

1. **공식 배터리**: sunlit **19.159**/interior **26.599** 양 게이트 PASS(예산
   19.2/26.84·봉인값 대비 +0.06 이내), 갤러리 `65d04ceca2c4dbff` 비트-동일, gdf_ao
   불변. **sc_viz 재시드**(bent 계약 = 의도된 콘텐츠 조명 변경; mean 4.49/max 56).
2. **히어로 배너 knob `P_SKYVIS_BENT_FLOOR`(기본 0 = 게이트 봉인 경로)**: 유효도
   계약만으로는 잔여 클래스(유효하나 수신자 법선과 직교하는 개구 방향 → dot≈0 암전)가
   AE 노출에서 여전히 사각으로 읽힘. 시각 최적이던 1/4-플로어 블렌드를 knob로 랜딩,
   **배너 레시피 전용**(AO_STRENGTH/AO_FLOOR knob 선례) — 기본 플립은 게이트 실측
   (+0.5)이 막음. 완전한 해소는 확률적-점유 트랙(§0 ①)의 몫.
3. **`P_GI_TRAPPED_FILL` 기본 OFF**: 주입 충전+all-rejected 소비 폴백(gdf_gi bit1·
   gdf_reflect bit27)은 E-측 실결함 수리로 실측 개선(바닥/잎 회수, bias −1.82→−0.74)
   이나 sunlit +0.23 — 자체 캘리브 패스 전까지 knob 잔존. knob-off는 변경 전 대비
   G/B ≤1 LSB 코드젠 드리프트(FP-민감 클래스, 정직 기록).
4. clippy `-D warnings` / fmt ✅.
