# F6M 계획서 — 나무·커튼 잔여 암부: 소비자 계약 3건 + PT 컷아웃 충실도

> 상태: 진행 중 (2026-07-20). 동기 = 사용자 보고: F6L 이후에도 히어로에서 **사이프러스
> 캐노피 내부 잎이 근흑**, **그늘측(우측 콜로네이드) 배너/커튼 주름이 흑화**. 본 페이즈는
> F6I §0 재시도-금지 원장의 어떤 항목도 재시도하지 않는다 — 랜딩 항목은 전부 (a) 신규
> 발견 소비자-계약 결함이거나 (b) PT **레퍼런스 자체**의 충실도 결함이다. 근본(확률적-점유
> 표현)은 §4에 사전 등록만 하고 다음 페이즈로 넘긴다.

## 1. 증거 — EV11 고정노출 배터리 (히어로 카메라, 단일-차이 17행)

부위별 평균 sRGB (R/G/B, 마지막 = 근흑 픽셀 비율). 대조군: `green_L`(양지 드레이프),

**판정 (재실험 금지):**

1. **PT 레퍼런스가 라스터보다 어둡다** — tree_dark에서 라스터가 PT의 **8배**, green_R도
   ×1.6. 근본 = **PT가 `alphaMode: MASK` 컷아웃을 알파-테스트하지 않음**(데칼만 확률
   통과, rt_path.slang) → 잎 카드 투명 텍셀이 전 레이(프라이머리/바운스/섀도)에 불투명
   → 사이프러스가 레퍼런스에서 검은 불투명 블롭. PT V-오라클도 블롭 외피를 잼(전 부위
   ~250 = 무효). **나무의 게이트-정직 목표치는 PT 수리 전에는 존재하지 않았다.**
2. **소비자 V 전역 붕괴 재확인**(F6H 불변): dv13 0-6.6/255, 양지 바닥조차 0. 출하 앰비언트
   전체가 crutch(tint 누설 0.15×lum×SKY_GAIN 6.0)로 운반되는 중.
3. **그늘측 석재 결손 ×1.47** = 진짜 라스터-대-PT 결손(stone_R 20.1 vs 29.6): 양지 바닥
   → 콜로네이드 벽의 **스톤-바운스 E가 기아**(dv15의 E는 총량 미미+과청 스펙트럼 B>R).
   우측 전체가 탁해져 커튼이 "너무 검게" 읽히는 지각적 주범. 수리는 점유 트랙 몫(§4).
4. **bent/dot_factor(V² 크러시) 실기여**: bent0에서 tree_dark +31%/red_R +12%. 직교-개구
   dot≈0 클래스(F6L 잔여)의 정량. 전면 해소는 점유 트랙 몫 — 소비측 스칼라 개입은 F6L
   수리 원장이 전부 게이트 기각(재시도 금지).
5. **AO 체인 배제**(dv9 균일 ~0.74, 플로어 상회), **fill1 이 뷰에선 미미**(+2% — F6L
   박스는 커튼 투영이었고 히어로 시점의 잔여 암부는 E-회수로는 안 열림).
6. **커튼 절대치는 PT 정합권**: 그늘 천의 "검음"의 상당 부분은 레퍼런스도 어두운 정직한
   어둠(직접광 0 + 바운스만). 지각 개선의 물리 근본 = ③(주변 석재)과 §4 반투명.

## 2. 랜딩 수리 (전부 원장-합법)

1. **C7 — 캐시-릴라이트에 F6L 유효도 계약 이식** (`sdf_cache_light.slang`): 밴드-1을
   무조건 재정규화하던 pre-F6L 패턴 잔존 → 프로듀서식 `(b/|b|)·sat(√3·c0/|b|)` +
   소비측 rel-페이드(pbr와 동일). 유효 셀은 바이트-동일, 노이즈만 bent-프리로.
2. **T4/C4 — gdf_gi·gdf_ao 뷰어측 법선 플립** (신규 발견): pbr는 back-face 법선을
   뷰어측으로 플립(`two_sided = !is_gallery`)하는데 같은 G버퍼 법선을 소비하는
   gdf_gi(V/E/bent 재구성)·gdf_ao(AO 마치)는 원시 법선 사용 — 양면 지오메트리(잎 카드·
   커튼 주름) back-face 픽셀이 반구를 거꾸로 평가. 수리 = 동일 계약의 플립(근평면
   언프로젝션으로 카메라 불요, gdf_gi flip_y bit2·gdf_ao flip_y bit1, 갤러리 비트-불변).
   현 스택에선 V 붕괴가 가려 효과 제한적이나, V가 정직해지는 순간 하중을 받는 계약.
3. **PT MASK 컷아웃 충실도** (`rt_path.slang`, `mesh.rs` params.z 재사용): 콘텐츠 히트
   테이블의 미사용 ao 슬롯에 `alpha_cutoff` 패킹, 전 레이 타입에서 컷아웃 텍셀 통과 —
   메인 루프는 **바운스 미소모 재시작**(V/E-오라클 bounce-1 tally 보존), NEE 섀도는
   `sun_blocked()` 재시작 루프. LOD 0 샘플(밉-평균 알파는 원거리 캐노피 용해), 스킵 캡
   32(초과 = 불투명 — 어두운 쪽 바이어스, 누광 없음), RNG 무소모(비-MASK 경로 스트림
   비트-동일). 갤러리/코넬 = `content_instances()` 게이트로 무변경.
4. **T2 — all-rejected 하드-블랙 폴백 디커플 (측정 시도)**: 점유-게이팅 트라이리니어의
   8코너 전기각 → E=0·V=0 하드-블랙 폴백(vol_occ bit1)이 `P_GI_TRAPPED_FILL`(기본
   OFF)에 결합돼 있음(main.rs). 단독 arm의 게이트 비용 측정 후 채택/철회. → 결과: (기입)

### 2b. 격리 실측 (EV11 히어로 카메라, 단일-차이)

- **C7 단독(`P_GDF_FACING_FLIP=0`) = base와 전 부위 소수점 일치** — 이 씬에선 캐시의
  bent-유효 텍셀이 지배적이라 계약 이식은 불활성(설계대로: 유효 셀 바이트-동일). 계약
  가치는 V-정직화(점유 트랙) 이후 발현되는 클래스.
- **플립 단독이 전 개선의 원천**: stone_R **+26%**(20.1→25.4, PT 29.6로 접근)·teal_R
  +17%·red_R +11%·tree_dark +6%, gdf_ao 천 부위 189→197-200. 그늘측 석재 raw E **×2.4**
  (5.3/9.5/13.5→13.4/20.8/24.0, 근흑 0.81→0.14).
- **2차 피드백 평형 이동(주의 기록)**: 바닥의 raw E가 파란 베일(과청 sky_fill 잔재)+빔
  자기-피드백 흰 패치에서 **국소 유색 바운스 구조**(적색 커튼·잔디 글로우)로 재분배 —
  바닥 픽셀은 front-face(플립 무대상)이므로 화면→캐시 lit-캘리브→볼륨 수입 루프의 평형
  이동. floor_sun 지역은 잔디(양면 카드) 혼입 대조군이라 +22% 상승에 플립 직접분 포함.
  판정은 공식 블록 게이트가 심판(§3).

### 2c. 게이트 판정 — 플립 기본 ON 기각 (F6H 클래스)

전체 스택(플립 ON) 공식 배터리: **갤러리 PASS(바이트-앵커 유지)** / sunlit 20.627(예산
19.2)·interior 32.569(26.84, bias +22.88) **FAIL** / sc_viz·gdf_ao SHA 변경. 판독:

- 플립은 그늘 **석재**를 PT 방향으로 올리지만(20.1→25.4 vs PT 29.6), 그늘 **천**은
  라스터가 이미 PT 과명 상태에서 더 벌린다(teal 21.2→24.8 vs PT 6.5). 현행 레퍼런스
  (천 반투명 없음) 하에서 net 악화 — **게이트가 옳게 기각**. F6H unsigned 심과 동일한
  "물리 수리가 크러치 캘리브를 개방" 클래스.
- 결정: **`P_GDF_FACING_FLIP` 기본 OFF**(코드 계약·노브 잔존), 히어로 레시피 옵트인
  (P_SKYVIS_BENT_FLOOR 선례). 기본 ON 재심 = §4 점유 트랙의 결합 재캘리브 시점, 또는
  천 반투명이 레퍼런스를 정직화한 뒤.
- 2-4(T2 디커플)는 본 페이즈 보류 — 점유 트랙의 α-가중 위트니스가 이진 위트니스 자체를
  대체할 예정이라 독립 측정 가치가 낮다.

## 3. 게이트 정책

- 갤러리 앵커: 전부 **바이트-불변 필수**(위 수리 전부 콘텐츠-게이트).
- PT 예산 게이트(sunlit 19.2 / interior 26.84): **③은 PT 레퍼런스 변경** — 잔차의 정의역
  자체가 바뀌므로 결과에 따라 재기준선 사유가 성립(선례: F6C tint 캘리브). 하향이면
  `--update` 래칫, 상향이면 본 문서에 사유·수치 기록 후 재시드.
- sc_viz: 라스터 소비자 변경(1·2)이 표면 캐시 lit에 닿으므로 재시드 가능성 있음 — 구조
  무변화 확인 후 재시드.

## 4. 사전 등록 — 후속 페이즈 (이것 없이 §0 트랙을 열지 말 것)

- **F6N: 확률적-점유 필드** (F6I §0 재개 조건 ① — 근본): 베이크 시 복셀 삼각형-커버리지
  → R8 α 채널(pack_like 아틀라스, ~+14MB; 340MB 천장 재시도 아님), **V·태양 가시성만**
  투과율 마치 `T *= (1-α)^(ds/vox)` — radiance는 첫-히트 유지(τ-radiance 기각 준수).
  착지 = sky_gain 6.0/tint 재캘리브 동반(F6G 재개방, t≈0.10 후보), 심 기본 ON 재심.
  §1-3(석재 바운스)·§1-4(V² 크러시)·dv13 붕괴가 전부 이 트랙에서 해소된다.
- **천·잎 반투명(translucency)**: 역광 천 글로우/잎 투과의 물리 근본. **PT 먼저**(thin
  BSDF — 레퍼런스 충실도), 이후 라스터 근사를 게이트로. 커튼 지각 품질의 최대 잔여 레버.
- **gdf_reflect 법선 플립**: T4/C4의 스페큘러 판(동클래스, 반사 방향 재구성).

## 5. 함정

- PT V-오라클은 ③ 수리 전 캡처와 비교 불가(블롭 외피 측정치).
- `P_GI_VOL_OCC=0`/`P_GI_VOLUME=0`의 밝아짐은 경로-전환 착시(F6L §2) — 판정에 쓰지 말 것.
- EV11 행렬과 AE 히어로는 도메인이 다름: 지각 판정은 AE, 귀속 판정은 EV11만.
- 히어로 레시피(level.rs 주석)는 AE + `P_SKYVIS_BENT_FLOOR=0.25` — 게이트 제외 knob.
