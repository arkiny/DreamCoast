# F4B2 계획서 — 볼륨-조건부 멀티바운스 부스트 프로브 (fine 스택의 산포-잔차 에너지 회수)

> 상태: **완료 — 기각(창 부재), 코드 무변경** (2026-07-15). 판정은 §4. 선행
> [phase-f4b-hierarchical-cache-plan.md](phase-f4b-hierarchical-cache-plan.md) §5 판정(기본 ON
> 보류: tint 창 공집합)의 후속 리드 ①. 스윕은 기존 `P_GI_MULTIBOUNCE` env만 사용 — 측정 프로브
> 예외 범주. 랜딩 커밋(창 존재 시 fine-조건부 기본값 ~5줄)만 게이트 뒤에 따른다.

## 0. 왜 (F4B의 미해결 잔차)

F4B 판정: fine 스택은 coarse의 바이어스 잔차를 산포 잔차로 바꾸며, **수렴 시 sunlit이
언더슈트로 역전**(lit 비율 coarse +7.8% → 수렴 fine −2.3%)한다 — 계통적 **에너지 결손**이
존재한다. 후보 손실원: SH-L1 재구성의 음수 클램프(gi_volume.slang eval_irradiance), occ-가중
재정규화의 전멸→0 경로, B2/V-게이팅 체인의 중첩 감쇠. tint는 (1−V)-가중 **중립** 필이라 이
결손을 채우면 색 앵커가 깨졌다(t0.2에서 lit 비율 1.082 복귀). **mb 부스트는 albedo-가중·
GI-비례**(pbr.slang:692-698: `gi /= max(1 − albedo·k, 0.25)`)라 밝은 석재의 바운스-리트
영역에 선별적으로 에너지를 돌려준다 — 다른 공간 시그니처로 (mb, tint) 2D 창이 열리는지가
이 프로브의 질문.

재해석 근거(leak §13b의 기각과 양립): 볼륨 경로 mb=0의 근거 "자체 피드백 = 무한 바운스라
기하급수 보상은 이중 계상"은 **무손실 재구성 가정**에서만 성립한다. F4B가 실측한 계통 손실이
있으면, 작은 k는 이중 계상이 아니라 **재구성 손실 보상**이다 — 값은 측정이 정한다(k가 크면
이중 계상 영역으로 넘어가 interior가 다시 과밝아질 것 — 그 경계가 창의 존재 여부).

## 1. 기지 사실 (F4B에서 — 재측정 금지)

- 공식 기준 행(fine+t0.15+mb0): sunlit **29.080**(budget 28.25) / interior **31.627**(31.91).
- tint 기울기(공식, t0.15↔0.2): sunlit **−18.0/unit**, interior **+8.28/unit**.
- coarse 기본 경로: 27.952/31.608 비트-재현, budget 래칫 28.25/31.91 — **이 페이즈에서 불변**
  (mb는 fine-조건부로만 랜딩; coarse의 mb=0 근거는 유지).
- 측정 규약: 공식 러너만(또는 같은-설정 raster+PT 쌍), EV11 분해는 AUTO_EXPOSURE=0, 셔머는
  CAPTURE_SEQ=8 STEP=0 동일 커맨드 A/B, 백그라운드 cd 명시, 장시간 런 caffeinate.

## 2. 적응적 스윕 설계 (풀 그리드 금지 — 런당 ~22분)

- **R1 (기울기 확정, 2런)**: fine+t0.15에서 `P_GI_MULTIBOUNCE` ∈ {0.15, 0.3} 공식 쌍 —
  mb=0 기지 행과 3점으로 sunlit/interior의 mb-기울기 도출. albedo≈0.5–0.7 가정 시 k=0.15는
  GI항 ×1.08–1.12, k=0.3은 ×1.18–1.27.
- **판정 수식**: 선형 모델로 (mb, tint) 평면에서 sunlit ≤28.25 ∩ interior ≤31.91 영역을 계산.
  공집합이면 즉시 기각(§4). 비공집합이면 **R2 (확인, 1–2런)**: 창 내부 점(마진 최대)을 공식
  쌍으로 확인 + EV11 크롭 색 앵커(B−R −0.6±0.2 유지) + lit 비율(1.0±0.03 근방 유지 — mb가
  비율을 1.08대로 되밀면 tint 실패의 반복일 뿐이므로 기각 사유).
- **R3 (랜딩 게이트, 창 확인 시)**: 기본값 코드 랜딩(fine-조건부 `P_GI_MULTIBOUNCE` 기본 +
  필요시 tint 동반 조정) → 풀 매니페스트(기본 경로 불변 확인) + CLIP=1 셔머 A/B + 갤러리.

## 3. 게이트 (판정 통과 조건 전부)

sunlit ≤28.25 **AND** interior ≤31.91 **AND** EV11 크롭 B−R ∈ [−0.8, −0.4] **AND** lit 비율이
mb=0 대비 악화(1.05 초과 복귀) 없음 **AND** 셔머 무회귀(mb는 소비측 정적 스칼라 — 중립 예상,
측정 확인) **AND** 기본(coarse) 경로 비트 불변.

## 4. 판정·기록

- 창 존재 → fine-조건부 기본 랜딩(단일 커밋), F4B §5 후속 리드 ① 해소 기록, 기본 ON 재도전은
  별도 판단(이 페이즈 스코프 밖 — 창이 있어도 기본 ON은 sunlit/interior 공식 쌍 재판정 필요).
- 창 부재 → 기각 기록(반복 금지 목록에 추가), 다음 리드 = box-half 공식 스윕 / 메트릭 분해.

### 판정 (2026-07-15, R1 3점 — 공식 러너, fine+t0.15)

| `P_GI_MULTIBOUNCE` | sunlit (budget 28.25) | interior (budget 31.91) |
|---|---|---|
| 0 (기지) | 29.080 | 31.627 |
| 0.15 | 29.054 | 31.646 |
| 0.30 | 29.030 | 31.672 |

**기각 — 창 부재.** 완벽 선형: sunlit 기울기 **−0.167/unit**(필요 이동 −0.80 → k≈4.7 —
공식 `max(1−albedo·k, 0.25)`의 클램프 영역 밖, 도달 전에 interior +0.7 FAIL), interior
+0.150/unit. mb 레버는 tint(−18.0/unit)보다 sunlit에 **~100× 약하면서** interior 트레이드
부호는 동일 — (mb, tint) 2D 창의 기하를 전혀 바꾸지 못한다. R2/R3 불필요(선형 3점으로 종결).

**물리 해석**: 192프레임 게이트 상태의 lit 비율은 이미 1.008 — 곱셈형 에너지 반환이 보정할
**바이어스가 없다**. masked_avg 잔차는 per-pixel 산포이고, 전역(albedo-가중이라도) 곱셈자는
산포를 줄이지 못한다(lit_mean_pt 78.8→78.5의 미동은 AE 재미터링). §0의 "재구성 손실 보상"
가설 자체가 과녁을 빗나감: 손실은 수렴 극한(384프레임 언더슈트)에서 실재하지만, 게이트가
재는 192프레임 상태는 손실이 드러나기 전이다.

**반복 금지 추가**: 볼륨 경로 mb 부스트를 fine 게이트 레버로 재시도(전 구간 선형·무효 실측).
다음 리드 = ② fine box half 공식 스윕, ③ 메트릭 분해 논의(F6) — 산포 잔차는 스칼라 레버가
아니라 **필드 품질 자체**(fine spp/보간 차수/SH 대역)나 **메트릭 정의**의 문제로 접근해야
한다는 것이 이 프로브의 결론.

## 5. 비목표

coarse 경로 mb 변경(이중 계상 근거 유지) / PT 수정 / budget 재기준선(fine은 opt-in — budget
불변) / march 경로 mb(0.6) 변경 / tint 창 재탐색 단독 반복(F4B에서 공집합 확정).
