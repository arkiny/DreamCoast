# F4B3 계획서 — fine box half 공식 스윕 (산포 잔차의 공간 스코프 레버)

> 상태: **진행 중 — 프로브(신규 env 노브 1개, 소비/업데이트 로직 무변경)** (2026-07-15). 선행
> [phase-f4b-hierarchical-cache-plan.md](phase-f4b-hierarchical-cache-plan.md) §5 후속 리드 ②,
> [phase-f4b2-mb-boost-probe-plan.md](phase-f4b2-mb-boost-probe-plan.md)(리드 ① 기각 — 스칼라
> 곱셈 레버 무효, 필드 품질/스코프로 접근하라는 결론)의 직접 후속.

## 0. 왜

F4B 판정: fine 스택의 sunlit FAIL(29.080)은 per-pixel **산포 잔차**이고 스칼라 레버(tint·mb)
로는 못 줄인다. box half는 스칼라 캘리브가 아니라 **공간 스코프 레버**다 — half를 줄이면:
1. **sunlit 화면의 fine-커버 면적 축소**: sunlit 카메라(0,2,0)의 box 경계가 half m에 위치 —
   7.25 m는 아트리움 코트 대부분을 fine이 덮는다(F4B에서 산포가 발생한 바로 그 영역).
   5 m면 중경 이후가 coarse로 남아 산포 유입 면적이 줄어든다.
2. **프로브 간격 조밀화**: 2·half/32 — 7.25 m→0.45 m, 5 m→0.31 m, 4 m→0.25 m. 같은 32³
   예산으로 더 정확한 근거리 필드(F4B2 결론의 "필드 품질" 방향).
경쟁 효과: interior의 깊은 크롭 개선(−0.20)도 커버 면적에 비례해 줄 수 있다 — interior
카메라(-14,2,0)의 5 m box는 x∈[-19,-9]로 근거리 주랑은 유지하나 원경 회수는 줄어든다.
두 게이트의 half-반응이 서로 다른 부호일 수 있어(각자 다른 기하) 2D (half, tint) 창이 tint
단독(공집합 확정)과 다르게 열릴 수 있는지가 질문.

## 1. 기지 사실 (재측정 금지)

- half=7.25(파생 기본), fine+t0.15+mb0: sunlit **29.080** / interior **31.627**
  (budget 28.25/31.91). tint 기울기: sunlit −18.0/unit·interior +8.28/unit(t0.15↔0.2 국소).
- mb 레버 기각(F4B2, 재시도 금지). coarse 기본 경로 27.952/31.608 비트-재현 — **이 페이즈
  불변**(노브는 CLIP=1 경로 안에서만 읽힘).
- 측정 규약: 공식 러너/같은-설정 쌍·EV11 분해·셔머 A/B·cd 명시·caffeinate — F4B §10 그대로.

## 2. 노브 (이 페이즈의 유일한 코드 변경)

`P_GI_FINE_HALF`(미터, clamp [2,16]) — 미설정 = 기존 파생 `(ext/6).clamp(4,12)` 그대로
(레거시 무변경). 데드존·복셀 스냅·fade 마진은 설치된 box에서 파생되므로 자동 추종.
CLIP=0 기본 경로는 노브를 읽지도 않는다(box-설치 블록 내부).

## 3. 적응적 스윕

- **R1 (1런)**: half=5.0, fine+t0.15 공식 쌍. 판정 분기:
  - sunlit이 유의 개선(<28.6)이고 interior가 budget 내 → **R2**: half {4.0, 6.0}으로 곡선
    맵핑, 필요시 (half, tint) 결합 창 계산 → 후보 확인 런.
  - sunlit 개선이 미미(≥28.9) → 커버-면적 가설 기각 방향 — half 10.0 역방향 1런으로 단조성
    확인 후 기각 기록(스코프 레버도 무효 = 산포는 fine 필드의 국소 품질 자체가 원인).
- **R3 (랜딩, 창 존재 시)**: 기본 half 변경(파생식 조정) + EV11 색 앵커 + 셔머 A/B + 풀
  매니페스트(기본 경로 불변) + 갤러리.

## 4. 게이트 (F4B2와 동일 구조)

sunlit ≤28.25 **AND** interior ≤31.91 **AND** EV11 크롭 B−R ∈ [−0.8, −0.4] **AND** lit 비율
악화 없음 **AND** 셔머 무회귀 **AND** 기본(coarse) 경로 비트 불변.

## 5. 비목표

tint 단독 재탐색(공집합 확정) / mb 재시도(F4B2 기각) / 32³ 해상도 변경(볼륨 예산 페이즈 밖) /
캐스케이드(F4B §9 미발동 유지) / PT 수정 / budget 재기준선.
